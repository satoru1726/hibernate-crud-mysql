# hibernate CRUD with MySQL
a simple CRUD application using Hibernate ORM and MySQL.

## tech stack
- java
- hibernate ORM
- mySql Database

## features
- create, read, update, delete operations using hibernate ORM
- configured using hibernate.cfg.xml
- definitons included [as comments in raw code] where ever needed, for better understanding

## how to run
1. clone the repo
2. update database credentials in `hibernate.cfg.xml`
3. run the app
